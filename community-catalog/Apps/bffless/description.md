BFFless [](https://github.com/bffless/ce/actions/workflows/main-release.yml) [](https://github.com/bffless/ce/actions/workflows/pr-tests.yml) A self-contained platform for hosting images and static assets generated from CI/CD pipelines. View build artifacts like screenshots, test reports, and other generated assets through an intuitive web interface.
