Open Home Foundation Matter Server The Open Home Foundation Matter Server is an officially certified Software Component to create a Matter controller. It serves as the foundation to provide Matter support to Home Assistant but its universal approach makes it suitable to be used in other projects too. This project implements a Matter Controller Server over WebSockets using the official Matter (formerly CHIP) SDK as a base and provides both a server and client implementation. > [!IMPORTANT] > We’re excited to share that the Matter Server is being rewritten on top of matter.js! 🎉 > This means that the current project has moved into **maintenance mode** — we’ll still fix important bugs if they come up, but all new features will land in the new project instead. > We’ll post a link as soon as the first **alpha version** is ready for you to try out. 🚀 The Open Home Foundation Matter Server software component is funded by Nabu Casa (a member of the CSA) and donated to The Open Home Foundation.
