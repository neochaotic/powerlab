Holesail Switchboard A web interface to manage multiple holesail servers and clients for peer-to-peer connections. Features - Create and manage multiple holesail servers - Create and manage multiple holesail clients - Real-time status monitoring (running, failed, disabled, initializing) - QR code generation for server connection URLs - Secure mode support for servers - Persistent configuration storage Quick Start Run directly with npx (no installation required): This will start the server and automatically open your browser.
